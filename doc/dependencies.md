Dependencies
============

These are the dependencies currently used by Tapyrus Core. You can find instructions for installing them in the `build-*.md` file for your platform.

|Dependency | Minimum required |
|----|----|
|Autoconf|2.60|
|Automake|1.13|
|Clang|6.0.1|
|GCC|9.4.0|
|Python (scripts, tests)|3.4|
